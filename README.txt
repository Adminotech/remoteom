Download installer:
- http://download.meshmoon.com/remoteom/RemoteOM-2.0.exe


windows usage
-----
- Using Qt (4.7.4)
- Download and install Qt http://qt-project.org/downloads
- Run Qt Creator
- Open .pro file in QtCreator

Running RemoteOM.exe from rel/debug folders
-----
- Copy images folder to project build folder (e.g. RemoteOM-build-desktop)
- Copy necessary dlls QtCore, Gui, Network, Xml, mingw and libgcc_s_dw2-1.dll to rel/debug folders
- Run RemoteOM.exe
